# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The objective is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL-based analysis, interactive dashboards, and business reporting.

The project follows a structured analytics process, making it suitable for showcasing practical data analysis skills.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data types and dataset structure.
* Checked for missing values and duplicates.

### 2. Exploratory Data Analysis (EDA)

* Generated descriptive statistics.
* Analyzed distributions and relationships.
* Created visualizations to identify trends and outliers.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into PostgreSQL, MySQL, and SQL Server.
* Executed SQL queries using:

  * SELECT
  * WHERE
  * GROUP BY
  * ORDER BY
  * JOIN
  * Aggregate functions
  * Window functions (where applicable)
* Extracted business insights from the database.

### 5. Dashboard Development

* Connected Power BI to the processed data.
* Created interactive dashboards.
* Added charts, KPIs, filters, and slicers.
* Designed visuals to support business decision-making.

### 6. Reporting

* Summarized key findings in a professional report.
* Highlighted major insights and recommendations.
* Created a presentation using Gamma to communicate results effectively.

## How to Run

1. Clone this repository.
2. Install the required Python libraries:

   * pandas
   * numpy
   * matplotlib
   * seaborn
3. Open the Jupyter Notebook.
4. Run the notebook cells in order.
5. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
6. Execute the SQL scripts.
7. Open the Power BI (.pbix) file to explore the dashboard.
8. Review the report and Gamma presentation for business insights.
